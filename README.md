Medical Knowledge Graph to Promote Rational Drug Use: Model Development and Performance Evaluation
============
Description
------
This repository contains the code and data for our work. In this work, we present the methods of constructing medical knowledge graph for driving the application of rational drug use. 
We first construct the domain ontology framework of knowledge graph via studying rational drug use domain concepts, authoritative books and drug instructions. 
And then we extract entities and relations based on the combination of two mechanisms. One is the supervised mechanism to train a named entity recognition model 
via deep learning technique, and another is the rule-based approach to extract relations between entities. In the last step we disambiguate and standardize the knowledge of entities
and relations to enrich the knowledge graph. We create and release the medical entity and relation annotation dataset that are manually labeled from raw texts. The experimental results verify the effectiveness of the proposed methods.
